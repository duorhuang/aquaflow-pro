# 出勤管理与性能优化需求

## 日期
2026-04-06

## 需求描述

### 1. 教练出勤管理权限
- 教练需要拥有完整的出勤管理权限
- 训练时教练可以查看有多少人到场、具体是谁来了
- 教练进行打卡操作，掌控训练出勤情况
- 得到完整的统计结果（到场人数、缺席人数、到场率）
- 队员不能自行打卡，避免虚假出勤

### 2. 系统性能优化
- 打开网页时数据需要快速加载
- 解决 15-20 秒等待的问题（Neon 数据库冷启动）
- 实施 keep-alive 机制保持数据库活跃

### 3. 中国大陆无 VPN 可用性
- 确保网站在没有 VPN 的情况下可以正常使用
- 确保在中国大陆各地都能顺畅访问

## 实施方案
1. 新建教练出勤管理页面 `/dashboard/attendance`
2. 移除运动员自助打卡功能（改为只读日历）
3. 添加 `/api/keep-alive` 端点用于定时驱动数据库
4. 验证 Cloudflare Pages 在中国大陆的可访问性
