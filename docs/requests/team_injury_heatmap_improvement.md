# 团队伤病热力图谱优化需求

## 需求描述
当前“团队伤病热力图谱”中的人体模型过于简陋（仅由几个简单形状连接而成，比例太小，显得奇怪），无法准确表达疼痛部位。

## 改进要求
需要改进人体模型的设计，至少需要清晰地标识出以下主要肌肉酸痛/伤病部位：
1. 肩部 (Shoulders)
2. 胸部 (Chest)
3. 颈部 (Neck)
4. 腰部 (Waist)
5. 腹部 (Abdomen)
6. 大腿和小腿（包含内侧区域）(Thighs and calves, including inner areas)
7. 手部 (Hands)

同时，需要优化整体的人体比例和大小，使其更自然且更易于观察。
