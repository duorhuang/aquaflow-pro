# 需求记录：男/女性别分化虚拟形象与 XP 双轨制经济体系 (2026-05-22)

## 需求背景
随着虚拟养成与商城系统的引入，为了提升运动员（特别是青少年队员）的参与感与真实代入感，需要对虚拟形象与经济系统进行精细化迭代。

## 核心需求明细

### 1. 虚拟形象与商城的男/女性别分化 (Gender-Differentiated Avatar)
- **性别划分**：
  - 男生：露出上身（游泳特征），默认穿着基础泳裤。
  - 女生：默认穿着基础连体泳衣。
- **穿戴槽位与共享/专属规则**：
  - **共享槽位**：包括 `头部 (泳帽)`、`眼部 (泳镜)`、`脚部 (脚蹼/拖鞋)`、`手持 (浮板/划水掌)`、`背景 (场景)`。男女队员看到和购买的商品完全相同，同一件装备在男女基底小人上的锚点与贴合位置需保持绝对一致，避免割裂。
  - **专属槽位**：包括 `身体 (泳衣/上衣)` 与 `下肢 (泳裤/裙装)`。
- **商城过滤**：
  - 商城将根据队员的性别进行自动过滤。男生队员不会看到或购买任何女生专属泳衣；女生队员同理，不会看到男生专属泳裤。
  - 200 件商城的商品中，约 130 件为男女共享，35 件为男生专属，35 件为女生专属。每个队员在商城中实际可见约 165 件商品。
- **入队礼包 (Starter Pack) 适配**：
  - 新队员注册时需要选择性别。
  - 男生赠送：基础白色泳帽 + 透明泳镜 + 黑色素泳裤 + 200 XP 初始余额。
  - 女生赠送：基础白色泳帽 + 透明泳镜 + 蓝色基础泳衣 + 200 XP 初始余额。

### 2. XP 双轨经济引擎 (XP Dual-Track System)
- **解决痛点**：避免队员因在商城消费 XP 购买装备而导致“等级下降/退步”的悖论。
- **双轨设计**：
  - **`totalXp`（总累计经验）**：只增不减，体现队员从“游泳新手”到“传奇冠军”的终身成长之路，用于决定当前的等级 (Lv.1 ~ Lv.8) 与等级称号。
  - **`balance`（可用余额）**：可增可减，专门作为商城消费货币。
- **账务处理**：
  - 获得 XP 时：`totalXp` 和 `balance` 同时增加相同数额。
  - 在商城消费购买商品时：仅扣除 `balance` 余额，`totalXp` 保持不变，因此队员的等级永远不会因为消费而降低。
