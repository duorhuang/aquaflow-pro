# 反馈系统重构需求

## 需求日期
2026-04-05

## 需求描述
将队员反馈页面拆分为两个独立的概念：

1. **队员反馈（训练日记/打卡）**：队员每天要写的反馈，记录日常训练感受
2. **训练反馈（教练周期性反馈）**：教练针对每个训练周期发出的，让队员做的反馈

### 教练端要求
1. 周期性反馈：
   - (a) 教练如何发送周期性反馈请求
   - (b) 教练如何收到队员的周期性反馈回复
2. 队员反馈：
   - (a) 教练如何收到队员每日、每周上传的反馈

### 改进方向
- 队员页面的反馈模块使用标签页区分两种反馈
- 让队员能够简单直观地理解每种反馈的用途
- 教练端页面同步调整，确保信息流畅通
