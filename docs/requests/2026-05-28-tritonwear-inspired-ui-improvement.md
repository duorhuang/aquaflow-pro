# Request: TritonWear-Inspired UI/UX Premium Redesign

**Date**: 2026-05-28  
**Reporter**: Product Owner / Swimmer Lead

## Background & Objectives
The user wants to elevate the visual and interactive design of **AquaFlow Pro** to match the modern, premium aesthetic of **TritonWear** (https://www.tritonwear.com/). 

The objective is to combine TritonWear's high-tech, performance-driven analytics vibe (electric teals/cyans, clean sans-serif typography, crisp data dashboards) with AquaFlow Pro's unique gamified features (XP ledger, levels, buddy systems, avatar shop, and custom backgrounds/animations) while keeping the entire experience strictly bound to a professional "swimming" theme.

## Core Directives
1. **Study TritonWear for Inspiration**:
   - High-fidelity dark mode with luminous accents (neon teals, electric cyans, smooth deep blues).
   - Clean, athletic, high-contrast typography (sans-serif optimized for numbers and telemetry).
   - Rounded components with subtle micro-borders, clean layout hierarchy, and minimal clutter.
   
2. **Preserve AquaFlow Pro's Unique Personality**:
   - Maintain the fun, interactive, and gamified nature of the platform (Streak indicators, XP trackers, Avatar shop, custom CSS themes for training blocks like sprint, aerobic, etc.).
   
3. **Enhance All Clickable Touchpoints & Fonts**:
   - Ensure the fonts are premium and highly readable across languages (incorporating Outfit/Inter or similar modern Google fonts).
   - Apply highly polished, delightful micro-animations to every button, tab, and card hover.
   - Refine active/pressed states, click ripple effects, and database-driven elements.

4. **Provide a Comprehensive Improvement Plan**:
   - Analyze existing system capabilities and design the detailed redesign path across all modules.
